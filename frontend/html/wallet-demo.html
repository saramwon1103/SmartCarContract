Wallet Connection Feature

Click the "Connect Wallet" button in the header to test the wallet connection modal.

Features:

  • Modern gradient button design matching MORENT's brand
  • Smooth modal animation with backdrop blur
  • Support for multiple wallets: Phantom, Slope, Solflare, MetaMask, WalletConnect
  • Expandable "More options" section
  • Light/Dark mode support
  • Responsive design for mobile devices
  • LocalStorage integration to remember connected wallet
  • Close modal with ESC key or clicking outside
